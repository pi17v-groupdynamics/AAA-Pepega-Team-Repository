О программе:
    Программа предназначена для шифрования и дешифрования файлов и текста.
    Шифрование реализовано с помощью шифра Цезаря.

Шифр Цезаря:
    Шифр Цезаря — это вид шифра подстановки, в котором каждый символ в открытом тексте заменяется символом, находящимся на некотором постоянном числе позиций левее или правее него в алфавите. Например, в шифре со сдвигом вправо на 3, А была бы заменена на Г, Б станет Д, и так далее.
    Шифр назван в честь римского полководца Гая Юлия Цезаря, использовавшего его для секретной переписки со своими генералами.

Функии:
    1. Шифрование текста - в программе предусмотрены 2 возможности шифрования текста:
        а. Текстовый - двигает символы по алфавиту или по порядку, если это числа;
        б. Бинарный - двигает биты по порядку.

    Текст также можно считать из файла и сохранить.

    2. Реализовано шифрование и дешифрование:
        а. Шифрование - использует ключ для смещения;
        б. Дешифрование - использует отрицательный ключ(Например: если ключ 5, то смещение будет на -5).

    3. Предаставлена возможность работы с текстом по диапазону ключей. Перед запуском фанкции необходимо выбрать файл для сохранения и указать диапазон([левый ключ] < [правый ключ]).

    4. Шифрование файлов - программа работает с файлами любого размера и расширения. Перед запуском необходимо выбрать зашифрованный файл или файл для шифрования. Результирующий файл будет создам в том же каталоге с приставкой "- encrypted" или "- decrepted" в зависимости от режима.
    ВНИМАНИЕ: Если прервать работу результирующий файл будет удалён!

Горячие клавиши:
    
    Работа с текстом:
      Shift + P : увеличить значение ключа на 1;
      Shift + M : уменьшить значение ключа на 1;
      Shift + O : выбрать файл для чтения;
      Shift + S : выбрать файл для сохранения;
      Shift + G : начало/отмена работы по диапазону;
      Shift + E : сменить режим на "зашифровать";
      Shift + D : сменить режим на "расшифровать";
      Shift + T : сменить режим на "Текстовый";
      Shift + B : сменить режим на "Бинарный";
      Shift + F : открыть окно шифрования файлов;
      Shift + H : открыть Руководство пользователя;
      Shift + C : отменить чтение из файла;
      Shift + Ctrl + S : открыть файл для записи результатов работы по диапазону.
    
    Работа с файлами:
      Shift + P : увеличить значение ключа на 1;
      Shift + M : уменьшить значение ключа на 1;
      Shift + E : сменить режим на "зашифровать";
      Shift + D : сменить режим на "расшифровать";
      Shift + O : выбрать файл для шифрования или дешифрования;
      Shift + S : начать/отменить работу.
